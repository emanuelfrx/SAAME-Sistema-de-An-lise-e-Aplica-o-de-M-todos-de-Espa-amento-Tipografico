# SAAME Typography Lab

A specialized web-based environment for typography experimentation, focusing on glyph metrics tuning (LSB/RSB), font analysis, and spacing methodologies.

## 🚀 Overview

SAAME Typography Lab provides a professional interface for type designers and enthusiasts to fine-tune font spacing using established methods like **Tracy** and **Sousa**. It allows for real-time visualization of glyph adjustments and deep analysis of spacing consistency.

## ✨ Features

- **Multi-Method Tuning**: Support for standard spacing workflows (Tracy & Sousa).
- **Real-time Glyph Visualization**: High-fidelity SVG rendering of glyphs with dynamic side-bearing adjustments.
- **Analysis View**: Comparative analysis of glyph spacing across alphabets, numbers, and punctuation.
- **Performance Optimized**: Built with React and Vite, utilizing memoized components for smooth live-tuning experiences.
- **Universal Font Support**: Compatible with standard web font metrics and custom font objects.

## 🛠️ Tech Stack

- **Framework**: React 18+ (with Hooks and Memoization)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animations**: Motion (Framer Motion)
- **Build Tool**: Vite

## 📥 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or pnpm

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
